# AI Virtual Mouse

A virtual mouse system that enables cursor control using hand gestures captured through a webcam, demonstrating real-time computer vision–based human–computer interaction.

## Features
- Cursor movement using index finger tracking
- Gesture-based left click
- Real-time hand landmark detection
- Works with a standard webcam

## Tech Stack
Python, OpenCV, MediaPipe, PyAutoGUI, NumPy

## Project Files
- aivirtualmouseproject.py – main application
- HandTrackingModule.py – hand tracking module

## How to Run
pip install opencv-python mediapipe pyautogui numpy
python aivirtualmouseproject.py

## Note
This is a prototype project and performance depends on lighting conditions and camera quality.
